### 수강신청 사이트 ###


# 1)프로젝트 소개 #

거의 php와 mariadb만을 이용해서 만든 수강신청 사이트입니다.

강의삭제, 강의만들기, 강의수정, 로그인, 로그아웃, 수강신청, 수강신청삭제, 수강신청시간표, 개인정보수정 기능들이 있습니다.,

# 2)사용한 기술 #

Php, mariaDb, css, html
